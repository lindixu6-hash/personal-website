# 🎨 莫兰迪色系 + 补线体风格 - 设计说明

## 🎨 莫兰迪色系（Morandi Colors）

### 色彩来源
莫兰迪色系源自意大利画家Giorgio Morande的作品，以其低饱和度、柔和、高级的配色著称。

## ✨ 莫兰迪色系特点

### 1. 低饱和度
- 所有颜色都是低饱和度（不刺眼）
- 柔和、舒适、不疲劳
- 适合长时间阅读

### 2. 高级感
- 不艳俗
- 有质感
- 显得克制和优雅

### 3. 柔和温暖
- 不冷不硬
- 温暖的灰调
- 给人亲切感

## 📐 补线体风格（Minimalist）

### 设计原则

#### 1. 大量留白
```
✅ 内容区域之间有大量空白
✅ 段落之间有明显的呼吸空间
✅ 不拥挤、不压抑
```

#### 2. 线条作为主要视觉元素
```
✅ 1px细线作为边框
✅ 细线作为分隔线
✅ 去掉了所有厚重的阴影
```

#### 3. 克制的装饰
```
✅ 去掉了泡泡、星星、闪烁等动画
✅ 去掉了emoji
✅ 保留了最基础的交互
```

#### 4. 精致的排版
```
✅ 字间距加大（letter-spacing: 0.02em）
✅ 行高增加（line-height: 1.8）
✅ 字重降低（font-weight: 300/400）
```

#### 5. 简洁的颜色
```
✅ 只用3-4种主色
✅ 所有颜色都是莫兰迪色
✅ 不使用高饱和度颜色
```

## 🎨 具体设计元素

### 配色方案

#### 背景色
- 页面背景：Ivory (#FFFFF0)
- 卡片背景：White (白色) 或 Cream (#FFFDD0)
- 特殊背景：Seafoam (#E0F2F1)

#### 文字色
- 正文：Dim Gray (#696969)
- 次要文字：Slate Gray (#708090)
- 标题：Dim Gray (#696969)
- 链接：Dim Gray (#696969)

#### 强调色
- 主要强调：Rose (#E8B4B8) - 玫瑰红，优雅
- 次要强调：Wisteria (#C9A8C8) - 紫藤色，高级
- 边框：Sage (#BCAIA3) - 鼠尾草绿

---

### 排版特点

#### 1. 字体选择
```css
font-family: 'Crimson Pro', 'Noto Sans SC', sans-serif;
```
- **Crimson Pro**：优雅的serif字体（衬线体）
- **Noto Sans SC**：现代的sans-serif字体（无衬线）
- 英文用衬线，中文用无衬线
- 整体既优雅又现代

#### 2. 字重
- 标题：font-weight: 300（轻）
- 正文：font-weight: 400（常规）
- 强调：font-weight: 500（中等）
- 避免font-weight: 600（太重）

#### 3. 字间距
- 标题：letter-spacing: 0.15em（更舒展）
- 正文：letter-spacing: 0.02em（微调）
- 链接：letter-spacing: 0.05em

#### 4. 行高
- 所有内容：line-height: 1.8（更透气）
- 不拥挤，易阅读

---

### 交互设计

#### 1. 悬停效果
```css
.skill-category:hover,
.project-card:hover,
.education-item:hover {
    border-color: var(--wisteria); /* 紫藤色 */
    transform: translateY(-2px); /* 微微上浮 */
    box-shadow: 0 8px 20px rgba(0, 0, 0, 0.06); /* 轻微阴影 */
}
```

#### 2. 链接交互
```css
a::after {
    width: 0;
    height: 1px;
    background: var(--sage);
    transition: width 0.3s ease;
}

a:hover::after {
    width: 100%; /* 下划线展开 */
}
```

#### 3. BGM按钮
```css
background: transparent;
border: 1px solid var(--sage);
color: var(--dim-gray);
```
- 极简设计，无背景色
- 细线边框
